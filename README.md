# 🌙 Noites Brancas – Site Interativo

Este projeto é um site interativo inspirado no livro Noites Brancas de Fiódor Dostoiévski.
A proposta é criar uma experiência visual e narrativa que represente a atmosfera melancólica e contemplativa da obra.

O site apresenta trechos e conceitos da história enquanto utiliza efeitos visuais e animações para envolver o usuário.

📖 Sobre o projeto

O site explora os principais momentos da narrativa:

O sonhador, personagem solitário que vive mais em seus sonhos do que na realidade.

O encontro com Nástienka, que traz esperança e companhia.

As quatro noites, onde a relação entre os dois se desenvolve.

Uma decisão final do usuário, que altera o clima emocional do site.

A experiência muda dependendo da escolha do usuário.

🎭 Interatividade narrativa

No final da página o usuário responde à pergunta:

“Você continuaria esperando?”

Dependendo da resposta:

✔ SIM (esperança / tristeza)

O clima fica mais melancólico

A chuva começa a cair

A lua permanece

A atmosfera fica mais escura

❌ NÃO (aceitação / superação)

A noite dá lugar ao entardecer

Surge um sol no céu

Aparece um horizonte iluminado

Partículas de poeira dourada criam um clima mais leve

Essa escolha representa duas interpretações possíveis da história.

✨ Efeitos visuais implementados

O site possui vários efeitos para criar uma atmosfera cinematográfica:

🌌 Fundo animado

Estrelas desenhadas com Canvas

Movimento parallax reagindo ao mouse

🌫 Atmosfera

Camada de névoa animada

Vinheta cinematográfica nas bordas

🌙 Elementos do céu

Lua com parallax seguindo o mouse

Sol animado no final feliz

Horizonte iluminado surgindo no entardecer

🌧 Clima dinâmico

Sistema de chuva gerada via JavaScript

Intensidade da chuva varia conforme o final

✍ Texto animado

Typewriter effect na introdução

Revelação das seções conforme o usuário rola a página

🛠 Tecnologias utilizadas

HTML5

CSS3

JavaScript (Vanilla)

Canvas API

Não foram utilizadas bibliotecas externas para animações.
